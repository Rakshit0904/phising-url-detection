# AI-Based Phishing & Malicious URL Detection using NLP

## Overview
This project is a Machine Learning-based system that detects phishing and malicious URLs using Natural Language Processing (NLP) techniques. It analyzes URL patterns and classifies them as **phishing** or **legitimate**.

---

## Features
- Detects phishing URLs in real-time
- Uses NLP techniques for text processing
- TF-IDF vectorization for feature extraction
- Machine Learning models for classification
- Flask-based web interface

---

## Technologies Used
- Python
- Scikit-learn
- NLTK
- Flask
- HTML, CSS

---

## Dataset
- Trained on **500K+ URL dataset**
- Contains both phishing and legitimate URLs

---

## How It Works
1. User enters a URL  
2. URL is cleaned and processed using NLP  
3. Converted into numerical features (TF-IDF)  
4. Machine Learning model predicts result  
5. Output shown as **Phishing** or **Safe**
